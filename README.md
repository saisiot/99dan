# 제하의 구구단왕 🎮

7세 어린이를 위한 재미있는 구구단 학습 PWA 게임입니다!

## ✨ 주요 기능

### 🎯 5가지 게임 모드
- **오늘의 도전**: 매일 새로운 문제 도전
- **단계별 학습**: 2단부터 9단까지 차근차근
- **약점 극복**: 틀린 문제 집중 연습
- **연습 모드**: 자유롭게 연습하기
- **완벽 도전**: 실수 없이 완벽하게!

### 🏆 동기부여 시스템
- **레이팅 시스템**: 실력에 따라 등급 상승 (브론즈 → 실버 → 골드 → 플래티넘 → 다이아몬드)
- **거리 여행**: 문제를 풀면서 전국 여행 (0km → 1,500km)
- **통계 시스템**: 정답률, 평균 시간, 연속 정답 등 상세 기록
- **가상 라이벌**: 컴퓨터 친구들과 경쟁

### 📱 PWA 기능
- 오프라인에서도 플레이 가능
- 홈 화면에 앱으로 설치
- 빠른 로딩과 부드러운 UX
- 데이터 자동 저장 및 동기화

## 🚀 시작하기

### 로컬에서 실행

1. **저장소 클론**
```bash
git clone <repository-url>
cd 99dan
```

2. **테스트 서버 실행**
```bash
node test-server.js
```

3. **브라우저에서 열기**
```
http://localhost:3000
```

### PWA 설치

#### Chrome (데스크톱/안드로이드)
1. http://localhost:3000 접속
2. 주소창 오른쪽의 "+" 아이콘 클릭
3. "설치" 버튼 클릭

#### Safari (iOS)
1. Safari에서 http://localhost:3000 접속
2. 공유 버튼 탭
3. "홈 화면에 추가" 선택

## 📁 프로젝트 구조

```
99dan/
├── index.html                 # 메인 HTML
├── manifest.json              # PWA 매니페스트
├── service-worker.js          # 서비스 워커 (오프라인 지원)
├── css/
│   └── style.css             # 스타일시트
├── js/
│   ├── app.js                # 앱 초기화
│   ├── game.js               # 게임 로직
│   ├── ui.js                 # UI 컨트롤
│   ├── storage.js            # 로컬 저장소
│   ├── statistics.js         # 통계 관리
│   ├── rating.js             # 레이팅 시스템
│   ├── analytics.js          # 분석 및 추천
│   ├── dataSync.js           # 데이터 동기화
│   ├── landmarks.js          # 거리 랜드마크
│   ├── virtualPlayers.js     # 가상 플레이어
│   └── modes/                # 게임 모드들
│       ├── daily.js
│       ├── stepByStep.js
│       ├── weakness.js
│       ├── practice.js
│       └── perfect.js
└── assets/
    └── icons/                # 앱 아이콘들
        ├── icon-192.png
        ├── icon-512.png
        └── icon.svg
```

## 🎮 게임 방법

1. **모드 선택**: 5가지 게임 모드 중 하나를 선택
2. **문제 풀기**: 구구단 문제를 보고 정답 입력
3. **결과 확인**: 정답/오답 즉시 확인
4. **레벨 업**: 문제를 풀면서 레이팅 상승
5. **여행 진행**: 정답마다 거리 증가로 전국 여행

## 🔧 개발 및 테스트

### 검증 실행
```bash
node test-validation.js
```

모든 필수 파일, manifest, service worker를 자동 검증합니다.

### 아이콘 재생성
```bash
node generate-icons.js
```

필요시 192x192, 512x512 PNG 아이콘을 재생성합니다.

## 📊 기술 스택

- **Vanilla JavaScript**: 프레임워크 없는 순수 JS
- **PWA**: Progressive Web App 기술
- **Service Worker**: 오프라인 지원 및 캐싱
- **LocalStorage**: 클라이언트 데이터 저장
- **CSS3**: 반응형 디자인 및 애니메이션

## 🎨 디자인 특징

- 7세 어린이를 위한 밝고 직관적인 UI
- 큰 버튼과 읽기 쉬운 글꼴
- 즉각적인 피드백 (정답/오답)
- 부드러운 애니메이션 효과
- 모바일 친화적 반응형 레이아웃

## 📈 학습 효과

- **반복 학습**: 약점 극복 모드로 취약한 부분 집중
- **동기부여**: 레이팅과 거리 여행으로 지속적인 흥미
- **경쟁심**: 가상 라이벌과의 비교로 자극
- **성취감**: 단계별 도전과 등급 상승
- **자기주도**: 원하는 모드를 자유롭게 선택
